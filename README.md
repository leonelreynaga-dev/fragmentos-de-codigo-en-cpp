# fragmentos-de-codigo-en-cpp

Este repositorio contiene fragmentos de código en C++ para práctica y aprendizaje.

## Archivos

### busqueda_secuencial.cpp
- Implementa búsqueda secuencial en un array
- Permite ingresar valores y buscar un número
- Enseña ciclos `while`, arrays y condiciones

### busqueda_binaria.cpp
- Implementa búsqueda binaria en un array ordenado
- Incluye ordenamiento de burbuja
- Practica arrays, ciclos y búsqueda eficiente

### gotoxy.cpp
- Demuestra cómo mover el cursor en la consola con `gotoxy`
- Ideal para interfaces de consola dinámicas
